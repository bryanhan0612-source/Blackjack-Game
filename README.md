# Blackjack-Game

A simple browser-based Blackjack game built using HTML, CSS, and JavaScript. The game follows standard Blackjack rules where the goal is to get as close to 21 as possible without going over

How to play: 
1. Open blackjack.html in any web browser.
2. You are dealt two cards; the dealer gets one visible card and one hidden card.
3. Choose an action:
- Hit: Draw another card
- Stay: End your turn
4. The dealer will draw cards until their total is 17 or higher.
5. The winner is determined based on Blackjack rules.
6. The game automatically resets after a short delay.
7. Every player has a balance of $500. Every bet is $10 and if you keep losing and go to $0, the game is over. 

Technologies Used
- HTML5
- CSS3
- JavaScript

Created by Bryan Han as a learning project to practice JavaScript logic, and DOM manipulation.
